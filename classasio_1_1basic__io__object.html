Detailed Description

template<typename IoObjectService, bool Movable = detail::service_has_move<IoObjectService>::value>
class asio::basic_io_object< IoObjectService, Movable >

Base class for all I/O objects.

Note
All I/O objects are non-copyable. However, when using C++0x, certain I/O objects do support move construction and move assignment.
