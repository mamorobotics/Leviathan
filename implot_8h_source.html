467// Double precision version of ImVec2 used by ImPlot. Extensible by end users.

469struct ImPlotPoint {
470 double x, y;
471 constexpr ImPlotPoint() : x(0.0), y(0.0) { }
472 constexpr ImPlotPoint(double _x, double _y) : x(_x), y(_y) { }

1053// Styling colors in ImPlot works similarly to styling colors in ImGui, but
1054// with one important difference. Like ImGui, all style colors are stored in an
1055// indexable array in ImPlotStyle. You can permanently modify these values through
1056// GetStyle().Colors, or temporarily modify them with Push/Pop functions below.
1057// However, by default all style colors in ImPlot default to a special color
1058// IMPLOT_AUTO_COL. The behavior of this color depends upon the style color to
1059// which it as applied:
1060//
1061// 1) For style colors associated with plot items (e.g. ImPlotCol_Line),
1062// IMPLOT_AUTO_COL tells ImPlot to color the item with the next unused
1063// color in the current colormap. Thus, every item will have a different
1064// color up to the number of colors in the colormap, at which point the
1065// colormap will roll over. For most use cases, you should not need to
1066// set these style colors to anything but IMPLOT_COL_AUTO; you are
1067// probably better off changing the current colormap. However, if you
1068// need to explicitly color a particular item you may either Push/Pop
1069// the style color around the item in question, or use the SetNextXXXStyle
1070// API below. If you permanently set one of these style colors to a specific
1071// color, or forget to call Pop, then all subsequent items will be styled
1072// with the color you set.
1073//
1074// 2) For style colors associated with plot styling (e.g. ImPlotCol_PlotBg),
1075// IMPLOT_AUTO_COL tells ImPlot to set that color from color data in your
1076// **ImGuiStyle**. The ImGuiCol_ that these style colors default to are
1077// detailed above, and in general have been mapped to produce plots visually
1078// consistent with your current ImGui style. Of course, you are free to
1079// manually set these colors to whatever you like, and further can Push/Pop
